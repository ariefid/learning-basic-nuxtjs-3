# Learning Basic NuxtJS 3

## About

The source code is for learning nuxtjs framework.
I'm using NuxtJS 3.

Warning! NuxtJS 3 is not recommended for production use.

## Requirements

NuxtJS 3 requires a minimum NodeJS version 14.16 LTS or above.
